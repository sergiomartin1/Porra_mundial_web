# 🎮 PORRA MUNDIAL 2026 - Guía de Usuario

## 🔐 Sistema de Login

La aplicación ahora tiene **autenticación por contraseña**. Cada usuario debe:
1. Seleccionar su nombre de usuario
2. Introducir su contraseña personal
3. Click en "Iniciar Sesión"
---

## 📊 Funcionalidades

### Para Usuarios Normales:

1. **Hacer Apuestas (1-X-2)**
   - Elige 1 si gana el equipo local
   - Elige X si es empate
   - Elige 2 si gana el equipo visitante
   - Click en "Guardar Apuestas" cuando termines

2. **Ver Todas las Apuestas**
   - Compara tus predicciones con las de otros usuarios
   - Ve el progreso de cada participante

3. **Ver Clasificación**
   - Tablas de grupos actualizadas en tiempo real
   - Basadas en los resultados que introduce el ADMIN
   - Primeros 2 de cada grupo en verde (clasificados)

4. **Ver Puntuación Usuarios**
   - Ranking de la porra
   - Puntos de cada usuario (1 punto por acierto)
   - Porcentaje de acierto
   - Medallas 🥇🥈🥉 para los 3 primeros

5. **Cerrar Sesión**
   - Importante para proteger tus apuestas
   - El siguiente usuario debe loguearse con su contraseña

---

### Para ADMIN:

1. **Introducir Resultados Reales**
   - Campos para goles de cada equipo
   - Ejemplo: México 3 - 1 Sudáfrica
   - Click en "Guardar" para cada partido
   - Los partidos con resultado se marcan en verde

2. **Actualización Automática**
   - La clasificación de grupos se actualiza automáticamente
   - Los puntos de usuarios se calculan en tiempo real

---

## 🏆 Sistema de Puntuación

### Clasificación de Grupos (Equipos):
- ✅ **Victoria**: 3 puntos
- ✅ **Empate**: 1 punto
- ❌ **Derrota**: 0 puntos

Criterios de desempate:
1. Puntos
2. Diferencia de goles
3. Goles a favor

### Puntuación de Usuarios (Porra):
- ✅ **Acierto** (1, X o 2 correcto): 1 punto
- ❌ **Fallo**: 0 puntos

El usuario con más aciertos gana la porra.

## 📱 Uso Recomendado

### Flujo Normal:
1. **Antes de cada partido**: Los usuarios hacen sus apuestas
2. **Después del partido**: El ADMIN introduce el resultado real
3. **Consulta**: Todos pueden ver la clasificación y el ranking actualizado

### Consejos:
- ✅ Completa todas tus apuestas antes de que empiecen los partidos
- ✅ Guarda frecuentemente (botón "Guardar Apuestas")
- ✅ Cierra sesión cuando termines
- ✅ El ADMIN debe ser responsable y actualizar resultados a tiempo

## ⚠️ Importante sobre localStorage

### ¿Qué es localStorage?
Los datos se guardan **en tu navegador**, no en un servidor central.

### Implicaciones:
- ✅ Funciona sin internet después de cargar la página
- ✅ Rápido y sin costes
- ⚠️ Cada navegador tiene sus propios datos
- ⚠️ Si cambias de dispositivo, no verás tus apuestas anteriores
- ⚠️ Si limpias los datos del navegador, se pierden las apuestas

### Recomendación:
- Usa **el mismo navegador y dispositivo** durante todo el mundial
- No limpies los datos del navegador durante la competición
- El ADMIN debe usar siempre el mismo navegador para introducir resultados

## 🆘 Solución de Problemas

### "Usuario o contraseña incorrectos"
- Verifica que escribiste la contraseña exactamente como está
- Las contraseñas distinguen mayúsculas/minúsculas
- Ejemplo: `MARVELDREAMER` ≠ `marveldreamer`

### "No veo los resultados que puso el ADMIN"
- Refresca la página (F5 o Ctrl+R)
- Verifica que el ADMIN guardó los resultados

### "Perdí mis apuestas"
- ¿Limpiaste el historial del navegador?
- ¿Estás usando el mismo navegador/dispositivo?
- ¿El mismo perfil de usuario del navegador?

### "No puedo hacer apuestas"
- ¿Iniciaste sesión correctamente?
- ¿Estás en modo ADMIN por error? (el ADMIN no hace apuestas, solo introduce resultados)
